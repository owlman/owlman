## 关于我

我是一名以 **计算机技术写作与系统性学习** 为核心的创作者。

我长期记录并整理以下内容：

- 计算机基础与工程实践的**学习路径**
- 编程语言、工具链、系统配置中的**取舍判断**
- 技术写作的方法论，以及**如何把零散经验组织成可复用知识**
- 技术之外，对创作、职业、长期主义的思考

我不追求快速产出，也不刻意迎合热点；  
更在意的是：**哪些内容在三年、五年后仍然成立。**
